## 外汇市场教训  
**失败的原因**  
1.手动交易。 手动交易导致无数次爆仓，爆仓总额达数十万美金，证明自己不适合手动交易    
2.使用高风险的EA。多次使用高风险EA，爆仓上万美元    
3.坚持使用优秀EA。  
4.等待大势。在小资金情况下就急不可耐的想要赚大钱，导致失去耐心，账号爆仓   

## 外汇市场方法论  
1.复制成功的人，成功的模式，成功的方法。
2.制定目标和计划。制定目标和计划，并坚决执行
3.狼性。为了猎物可以蹲守一整天，一旦猎物出现快、准、狠的将它猎杀
