<h4>投资、交易、投机的区别</h4>
投资是对一个事物的认可，看好这个事物的未来，只关注未来价值，不关注短期的亏损。  

交易是交易员付出时间和精力去观察市场行情，并寻找良好时机，以较低的亏损去搏取较大的利润。  

投机是在市场中利用马丁格尔、平均成本、剥头皮等策略赚钱微薄利润的方式。


<h4>什么是交易</h4>
交易是一份职业。需要付出时间和精力去观察市场。如华尔街的交易员。
交易的灵魂就是寻找时机。
交易需要优化盈亏比，以较小的亏损搏取较大的利润。
交易需要严格控制仓位。
