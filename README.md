没有一个交易系统可以每天都赚钱，也没有一个交易系统
可以适合所有的行情。所以我们要做的就是要确定自己的
交易系统，只交易特定的行情。对于做趋势的系统而言，
利润永远来源于大的趋势，而不是想着从每日的波动中去
剥头皮，趋势交易系统交易越少越好，甚至一年或几年才
交易一两次  

谈趋势系统  
趋势系统就是要做趋势，避免调整，不参与调整。一旦调整
立马出局，空仓等待并尝试下一次趋势。
